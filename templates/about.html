How Worksheet Studio works

A quick reference for everything the dashboard supports.

Question types

Multiple Choice, Fill in the Blanks, Match the Column, Short Questions, and Detailed Questions — mix any combination, in any quantity.

Grade-aware themes

Grade 1–5 renders colorful and cartoonic with friendly icons. Grade 6–10 renders clean and professional. You can force either from School Setup.

Accurate answer keys

Every question type carries its own answer, so the key always matches what's printed — even when match-the-column options are shuffled per copy.

Layout controls

Questions per page, spacing, number of copies, same-order or shuffled-per-copy randomization, and a binding gap for printing.

Bilingual support

Full Urdu support using pure Urdu script (never Roman Urdu), with correct right-to-left shaping on every label and question.

Export

Download as PDF or Word (.docx). When an answer key is generated separately, both files are bundled into one .zip download.

Running this app

This dashboard is a Flask app. To run it locally:

pip install -r requirements.txt
python app.py

Then open http://localhost:5000. PDF export uses WeasyPrint, which needs Pango/Cairo system libraries installed — see README.md for platform-specific setup notes.
